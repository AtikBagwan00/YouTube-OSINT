# YouTube OSINT Advanced Extractor

A powerful Tampermonkey userscript that extracts hidden OSINT data from YouTube videos for research and analysis purposes.

## Features

- **Video Metadata**: Video ID, upload date, duration, category
- **Engagement Metrics**: Exact view counts, likes, dislikes (via API), comments
- **Channel Information**: Channel ID, subscriber count, handle
- **Advanced Analytics**: Engagement rates, video age, sentiment analysis
- **Hidden Data**: Tags, quality settings, family-safe status
- **One-Click Export**: Copy all extracted data to clipboard

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Click [here](youtube-osint-extractor.user.js) to install the script
3. Visit any YouTube video page
4. Click the red "OSINT Extract" button (bottom-right corner)

## What Data is Extracted

- Video ID & Channel ID
- Exact view counts (not rounded)
- Like/Dislike counts (includes API data)
- Upload date & video age
- Engagement rates & sentiment
- Tags, category, duration
- Channel subscriber count
- HD thumbnail URLs
- Quality settings & live status

## Usage

1. Navigate to any YouTube video
2. Click the "OSINT Extract" button
3. View extracted data in the popup panel
4. Use "Copy All Data" to export information
